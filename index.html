Steady as he codes...
My Mission
To make coding my second language and increase proficiency at front-end web development across many different industries ranging from the beer/beverage industry, skilled trades, eCommerce, and many more. I'm working towards my degree in Graphic Information Technology: Full Stack Web Development so that I can cover full range projects for all customers.
Skills

Structuring the web's content using modern semantic HTML.

Presenting beautiful web designs and interactions using modern CSS.

Rapidly developing responsive websites using Bootstrap, the web's most popular framework.

Prototyping front-end concepts with figma and bringing them to life.
